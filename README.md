# BPV Examen - Veiligheidscheck App

## Projectomschrijving
Dit is mijn examen project Software Development niveau 4

De applicatie is een webapplicatie voor het registreren van veiligheidschecks. Medewerkers kunnen inloggen, een checklist invullen en hun oude checklists bekijken. Een admin kan alle ingevulde checklists bekijken en controleren.

## Gebruikersrollen
- Medewerker
- Admin

## Belangrijkste functionaliteiten
- Login voor medewerker en admin
- Checklistformulier invullen
- Checklist opslaan
- Geschiedenis van ingevulde checklists bekijken
- Admin overzicht van alle checklists
- Detailpagina voor checklistgegevens

## Gebruikte technieken
- Vue.js
- JavaScript
- JSON
- GitHub

## Definition of Done
Een taak is klaar wanneer:
- De functionaliteit werkt zoals beschreven in de userstory.
- De acceptatiecriteria zijn gecontroleerd.
- De code netjes en overzichtelijk is geschreven.
- Data correct wordt opgeslagen en opgehaald.
- De functie getest is.
- Er bewijs is zoals een screenshots of testresultaten.
